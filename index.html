cascading style sheets

cascading style sheets,is a style sheet language used to describe the visual presentation of a document written in html. css describes how elements should be rendered on screen. it enables wed designers to control the layout, colors,font,and other visual aspects of multiple wed pages all at once.


how to insert css into html(3ways)

  • inline styking
  • internal styling
  • external styling

inline styling:-

in inline styling css styles directly apply to the individual html elements using the style attributes.


exmple:-h1 tag style="color;text-align:center;" it's come to the center



internal styling:-

internal styling in css refers to the method of embedding css rules directly within the html document using the style tag.

used by style tag body { background color-some color name } h1 { color-some color; margin-left;40px }

external styling

external css is used to style multiple html pages with a single style sheet. external css contains a separate css file with a .css extension.

how to link external stylesheet into html document

exmple link tag rel=stylesheet-href=some css file name

how to create external stylesheet

create a file in a folder with .css extension and write the css code.

specificity

inline style have higher specificity compared to internal and external styles



css cascading and importance:-

the cascading nature of css is designed to ensure that more specific rules take precedence. inline styles are considered the most specicific because they apply directly to the element.
